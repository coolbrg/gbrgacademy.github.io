What is Back End?

It refers to the work that happens in the Data Access layer of a piece of Software.

What is Back End Web Developement?

It refers to the portion of the Web Application which is not visible to users and ensures everything on the client-side actually works. Making updates and changes in addition to monitoring functionality of the site will be your primary responsibility. It usually consists of three parts; a server, an application, and a database.
